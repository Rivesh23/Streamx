# Netflix - Local Streaming Platform

A private, local Netflix-style streaming application.

## Features
- Auto-scan local Movies & TV Shows.
- TMDB Metadata integration.
- Dark mode UI.
- Local playback tracking.
- Electron desktop app wrapper.

## Setup
1. Install Python dependencies: `pip install -r backend/requirements.txt`
2. Run Backend: `python backend/main.py`
3. Run Electron: `cd electron && npm install && npm start`
